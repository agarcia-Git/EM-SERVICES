 # **TOTOLIST** — Application de gestion de tâches

***TOTOLIST*** EST UN SYSTEME PERMETTANT LE SUIVI DES TACHES.


🚀  **Fonctionnalités**

- ***Page d'accueil*** — Saisie du prénom avec validation
- ***Page des tâches*** — Affichage, ajout de tâches 
- ***Page de détail*** — Voir, marquer comme terminée, réouvrir ou supprimer une tâche
- ***Page statistiques*** — Vue synthétique : total, terminées, à faire, progression
  

**Technologies** 
HTML/CSS/JavaScript
